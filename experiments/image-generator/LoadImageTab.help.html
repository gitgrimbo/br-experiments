This section provides the ability to load an SVG image from locations such as:

  • a pre-defined source (this website)
  • any public URL
  • Google Photos

If you want to be able to set custom data/values in the SVG, then the SVG must have elements with ids. These ids are used by the site to pick out specific elements in the SVG.

Currently, simple text values can be set, such as players names and image URLs.

When you are happy with the image, press the CREATE PNG button to create a PNG version of the SVG. This PNG can then be used in social media posts.
