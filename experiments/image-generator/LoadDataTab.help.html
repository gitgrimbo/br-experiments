This section provides the ability to load data from external sources, such as Google Sheets.

Some SVGs have embedded data (that the author of the SVG has embedded), and that will also appear here. Embedded data is set in a <text> element in the SVG with id="sample.data". Embedded data is in JSON format and has the form "data-name": ["list", "of", "items"], where you can have multiple data-names for different data. For example: 

<svg>
...
  <text id="sample.data">{
    "data.player.position":[
      "C","P","1B","2B","3B",
      "SS","LF","CF","RF"
    ],
    "data.player.name":[
      "Rich Green",
      "Miguel Ju√°rez",
      "Nestor Martinez"
    ]
  }</text>
...
</svg>

The loaded data can then be used to set values within the SVG image.

Once the data is loaded you will be able to view it all below. But you need to use the SET IMAGE DATA tab to actually use the data.
